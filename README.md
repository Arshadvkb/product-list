# PRODUCT LIST

## TECH STACK
 - React
 - vite
 - javascript
 - tailwind

## PACKAGES
- react-router-dom
- axios
- zustand

## TOPICS COVERD
- Fetch data from a public api
- Search item by name 
- filter by price 
- state manged using zustand
- passing id through props

## SETUP
```
git clone https://github.com/Arshadvkb/product-list.git
cd product-list
```

## RUN
```
npm install
npm run dev
```
